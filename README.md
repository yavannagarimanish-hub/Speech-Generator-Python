# 🗣️ Neutral Speech Generator (Python)

A simple Python-based command line application that generates structured speeches based on topic, audience, tone, and length.

## 🚀 Features

- Multiple tones:
  - Formal
  - Informal
  - Inspirational
- Adjustable speech length
- Audience customization
- Clean formatted output
